# Auto Task

这是一个基于go的全自动可视化化HTTP工作流系统

刚学go，先抄一下`Cloudreve`的结构

## 目标

1. 以工作流的方式执行HTTP请求
2. 记录工作流执行情况

## 场景举例（预先实现的场景）

1. 哔哩哔哩漫画签到
2. 其乐社区自动登录
3. ...

## 额外信息
